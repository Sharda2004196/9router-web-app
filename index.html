Quick Start

Setup in 3 Steps

1

Install 9Router

Requires Node.js 18+. Install globally with npm or use npx.

npm install -g 9router
# or run without installing:
npx 9router
2

Start & Connect Providers

Run 9router, then open the dashboard at localhost:20128. Go to Providers → connect the free ones first: Gemini CLI (Google OAuth), iFlow (OAuth), Kiro (AWS Builder ID), Qwen (device auth). Each takes under 2 minutes.

9router
# Dashboard opens at: http://localhost:20128
# Connect providers from the Providers tab
3

Configure Claude Code

Set these two environment variables in your shell profile (~/.zshrc, ~/.bashrc, or ~/.config/fish/config.fish). The auth token value doesn't matter — 9Router handles real auth.

Mac/Linux:

export ANTHROPIC_BASE_URL="http://localhost:20128/v1"
export ANTHROPIC_AUTH_TOKEN="9router"

Windows (PowerShell):

$env:ANTHROPIC_BASE_URL="http://localhost:20128/v1"
$env:ANTHROPIC_AUTH_TOKEN="9router"

Windows (CMD):

set ANTHROPIC_BASE_URL=http://localhost:20128/v1
set ANTHROPIC_AUTH_TOKEN=9router

Windows users on WSL2 can use the Mac/Linux commands above.

Tip: 9Router's dashboard at localhost:20128 shows live quota usage, estimated savings, and which provider handled each request.

Configurations

Combos

A combo is an ordered fallback list. 9Router tries provider 1 first, falls back to 2 when quota runs out, then 3. Set combos in the Dashboard → Combos tab.

free-forever $0 / month

Pure free tier. Production-ready models, zero cost.
  1. gc/gemini-3-flash → 180K free completions/month (resets monthly)
  2. if/kimi-k2-thinking → unlimited free via iFlow
  3. qw/qwen3-coder-plus → unlimited free via Qwen

maximize-claude ~$25 / month

Get the most out of a Claude Code subscription with free backups so you never hit a wall mid-session.
  1. cc/claude-opus-4-6 → your Claude Code subscription (best quality)
  2. if/kimi-k2-thinking → free fallback when 5-hour quota resets
  3. qw/qwen3-coder-plus → unlimited free emergency fallback

always-on $20–$40 / month

Maximum uptime. Five fallback layers mean you never stop coding.
  1. cc/claude-opus-4-6 → Claude subscription
  2. gc/gemini-2.5-pro → Gemini CLI (free tier first)
  3. if/kimi-k2-thinking → iFlow unlimited
  4. qw/qwen3-coder-plus → Qwen unlimited
  5. kiro/claude-opus-4-6 → Kiro (AWS Builder ID, free Claude) (50/month after first month)
